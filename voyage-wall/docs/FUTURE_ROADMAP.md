# Future Roadmap

## Phase 1: MVP Launch

- Couple-branded guest entry.
- QR-based access.
- Photo and message upload.
- Shared Love Wall.
- Basic storage and data model.
- Mobile-first polished design.

## Phase 2: Moderation And Admin

- Admin dashboard.
- Pending/approved/hidden queues.
- Bulk approve.
- Delete and restore.
- Reported content handling.
- Event settings.
- QR code download.

## Phase 3: Personalization

- Theme presets.
- Custom couple names and monogram.
- Custom hero image.
- Color palette options.
- Decorative motif options.
- Welcome copy customization.

## Phase 4: Display Mode

- Projector/TV Love Wall.
- Auto-rotating memories.
- New memory highlight.
- Fullscreen mode.
- Reduced-control public display URL.

## Phase 5: Post-Wedding Keepsake

- Download all memories.
- Export message archive.
- Keepsake PDF.
- Slideshow generation.
- Couple-only archive link.
- Anniversary resurfacing.

## Phase 6: Guest Engagement

- Heart/react to memories.
- Featured memory.
- Prompt cards such as "Best dance floor moment" or "Advice for the couple".
- Guest photo challenges.
- Optional guest tagging.

## Phase 7: Operational Expansion

- Multi-event dashboard.
- Planner/vendor accounts.
- Package tiers.
- Event templates.
- Analytics.
- White-label venue offering.

## Phase 8: Advanced Intelligence

- Duplicate photo detection.
- Image quality filtering.
- Smart album grouping.
- AI-generated captions or summaries.
- Content safety assistance.

## Long-Term Vision

Voyage Wall can grow from a simple memory upload site into a premium wedding keepsake platform: live during the event, beautiful after the event, and emotionally valuable years later.
